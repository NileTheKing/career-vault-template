# 온보딩 프롬프트

아래 텍스트를 **그대로 복사해서** Claude Code에 붙여넣으면 된다.
Claude가 인터뷰를 진행하며 나의 무기 노트를 자동으로 생성해준다.

---

## 붙여넣기용 프롬프트

```
안녕. 나는 지금 취업 지원을 위한 Obsidian vault를 세팅 중이야.
CLAUDE.md 읽고 구조 파악해줘.

그 다음 나를 인터뷰해서 내 경험을 원자화 무기 노트로 만들어줘.
아래 순서대로 진행해:

1. 내가 진행한 프로젝트를 하나씩 물어봐. 프로젝트마다:
   - 어떤 문제를 해결했는지
   - 어떤 기술을 썼는지
   - 실측 수치가 있는지 (성능 개선, 에러율, RPS 등)
   - 내가 직접 구현한 부분이 뭔지
   각 항목을 파악한 뒤, T{번호}_{기술명}.md 파일로 projects/ 하위에 저장해줘.
   templates/무기_노트_템플릿.md 포맷 따를 것.

2. 프로젝트 외 경험도 물어봐. (인턴, 수상, 학내활동, 자격증 등)
   E{번호}_{경험명}.md 파일로 me/경험/ 하위에 저장해줘.
   templates/경험_노트_템플릿.md 포맷 따를 것.

3. 다 끝나면 _meta/무기_마스터_목록.md 인덱스 채워줘.

인터뷰는 한 번에 하나씩 질문해. 한꺼번에 여러 개 묻지 말 것.
내가 답하면 바로 파일로 저장하고 다음 질문으로 넘어가.
```

---

## 인터뷰 후 확인사항

- `projects/` 에 T1~Tn 파일 생성됐는지 확인
- `me/경험/` 에 E1~En 파일 생성됐는지 확인
- `_meta/무기_마스터_목록.md` 인덱스 업데이트됐는지 확인
- Obsidian에서 노트 열어서 Smart Connections 임베딩 완료 확인 (우측 패널)

---

## 무기 추가할 때

새 프로젝트나 경험이 생기면:

```
새 무기 추가해줘. [경험 내용 설명]
```

Claude가 기존 번호 이어서 파일 생성 + 마스터 목록 업데이트.
